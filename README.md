# Project THA

This is my first game project, built from scratch using HTML, CSS, and JavaScript.  
It’s a simple 3D shooter experience designed to run smoothly on both desktop and mobile devices.

---

## 🎮 About the Game

Project THA is a lightweight action game where the player moves through a 3D environment, interacts with enemies, and survives as long as possible.  
The game features:

- 3D graphics using the Three.js library  
- Touch and keyboard controls  
- Dynamic enemy behavior  
- A simple HUD for health and score tracking  

---

## 📱 How to Play

- Use the **left joystick** (or WASD keys) to move  
- Use the **right area** (or mouse) to look around  
- Tap the **fire button** (or press F) to shoot  
- Use the **reload button** (or press R) to refill ammo  

---

## 🛠️ Build & Deployment

The game is built using vanilla JavaScript and runs in any modern browser.  
The APK version was created using web-to-APK conversion tools, so it can be installed on Android devices.
